# 云服务器产品文档

## 支持的操作系统

### Linux 发行版

- Ubuntu 22.04 / 20.04 / 18.04
- CentOS 7 / 8
- Debian 11 / 10
- Rocky Linux 8 / 9
- AlmaLinux 8 / 9

### Windows Server

- Windows Server 2022
- Windows Server 2019
- Windows Server 2016

### 注意事项

1. 不同地域支持的操作系统可能略有差异
2. 部分操作系统需要额外授权费用
3. 建议使用 LTS（长期支持）版本

## 计费方式

### 按量付费

- 按秒计费，小时结算
- 随时创建和释放
- 适合短期需求和测试环境

### 包年包月

- 提前支付，享受折扣
- 1年付享 7 折优惠
- 适合长期稳定业务

### 节省计划

- 承诺消费换取折扣
- 灵活性与成本优化兼顾
- 适合可预测的工作负载

## 配置升级

### 在线升级

以下配置支持在线升级，无需停机：
- CPU 核心数
- 内存大小
- 数据盘容量

### 升级限制

- 操作系统不支持降级
- 部分规格不支持横向变更
- 建议在业务低峰期操作

### 操作步骤

1. 登录控制台
2. 选择目标云服务器
3. 点击"升级配置"
4. 选择目标规格
5. 确认价格并支付
