# 对象存储服务文档

## 存储费用

### 计费项

对象存储（OSS）费用由以下部分组成：

| 计费项 | 按量付费 | 包年包月 |
|--------|----------|----------|
| 存储容量 | 0.12 元/GB/月 | 享受折扣 |
| 请求次数 | 0.01 元/万次 | 包含额度 |
| 流量费用 | 0.5 元/GB | 包含额度 |

### 存储类型

| 类型 | 适用场景 | 价格 |
|------|----------|------|
| 标准存储 | 频繁访问 | 基准价 |
| 低频存储 | 每月访问 < 3 次 | 约 0.08 元/GB/月 |
| 归档存储 | 极冷数据 | 约 0.033 元/GB/月 |

### 成本优化建议

1. 使用生命周期策略自动转储
2. 选择合适的存储类型
3. 开启请求级别收费提醒

## 访问权限控制（ACL）

### ACL 类型

| 权限 | 描述 | 适用场景 |
|------|------|----------|
| private | 仅所有者可读写 | 私有数据 |
| public-read | 所有人可读 | 静态网站、CDN 源站 |
| public-read-write | 所有人可读写 | 不推荐 |
| bucket-owner-read | 桶拥有者可读 | 跨账号访问 |

### 权限设置

#### 控制台设置

1. 进入对象存储控制台
2. 选择目标 Bucket
3. 点击"权限管理"
4. 配置 ACL 或 Bucket Policy

#### API 设置

```json
{
  "acl": "private",
  "policy": {
    "Version": "2012-10-17",
    "Statement": [
      {
        "Effect": "Allow",
        "Principal": {"AWS": ["arn:aws:iam::123456789:root"]},
        "Action": ["oss:GetObject"],
        "Resource": "acs:oss:*:*:example-bucket/*"
      }
    ]
  }
}
```
