# 密钥管理服务文档

## 密钥管理概述

密钥管理服务（KMS）提供安全的密钥托管和加密解决方案。

## 密钥类型

### 对称密钥

- 算法：AES-256, SM4
- 用途：数据加密
- 托管方式：云端托管 /  BYOK

### 非对称密钥

- 算法：RSA-2048, SM2
- 用途：数字签名、证书
- 支持硬件安全模块（HSM）

## 密钥生命周期

```
创建 → 启用 → 正常使用 → 轮换 → 禁用 → 删除
```

## 管理功能

### 密钥创建

```json
{
  "key_id": "kms-123456",
  "key_usage": "ENCRYPT_DECRYPT",
  "key_spec": "AES_256",
  "description": "数据库加密密钥"
}
```

### 密钥轮换

- 支持自动轮换（按年/月）
- 保留历史密钥用于解密
- 记录轮换审计日志

### 权限控制

基于 IAM 的细粒度权限控制：
- `kms:CreateKey` - 创建密钥
- `kms:Encrypt` - 加密数据
- `kms:Decrypt` - 解密数据
- `kms:ScheduleKeyDeletion` - 删除密钥
